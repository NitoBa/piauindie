## 🚀 How to install

To install the depencies for all apps (web and admin) use the following script

```bash
npm run install:all
```

To install the depencies for each app (web and admin) use the following script

Web:

```bash
npm run install:web
```

or

Admin:

```bash
npm run install:admin
```
